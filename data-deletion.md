# Data Deletion Policy for Bookshelf

**Effective Date:** February 19, 2026

This page explains how to request deletion of account-related personal data for Bookshelf.

## 1. What Can Be Deleted

Upon verified request, we will delete or anonymize, where applicable:

- account profile data (email, display name, user ID),
- synced user library state (favorite/read/saved),
- stored premium metadata associated with your profile.

## 2. How to Request Deletion

Send your request to: **bookshelf.developer@gmail.com**

Use this subject line:  
`Data Deletion Request - Bookshelf`

Include:

- your account email,
- your user ID (if available),
- confirmation that you want permanent deletion.

## 3. Processing Time

Verified requests are typically processed within **30 days**.

## 4. Limited Retention Exceptions

Some records may be retained where required for:

- legal compliance,
- security and fraud prevention,
- financial/accounting obligations,
- dispute handling.
